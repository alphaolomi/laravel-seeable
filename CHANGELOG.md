# Changelog

## [0.3.0] - 2022-07-01

### Changed

- Rename scopes `seenLastX` to `seenPastX`.

## [0.2.2] - 2022-07-01

### Fixed

- Check if current seen timestamp is `null` before comparing.

## [0.2.1] - 2022-07-01

### Fixed

- Check if user has concern in middleware.

## [0.2.0] - 2022-07-01

### Changed

- Remove `where` prefix from scopes.

## [0.1.0] - 2022-06-30

### Added

- Seeable concern.
- SeeUser middleware.
- Query scopes.
